# 🧠 Mini Balança Inteligente

Sistema em Python que simula uma balança inteligente capaz de ler pesos e classificar produtos automaticamente.

![Demo](assets/demo.png)

---

## 🚀 Sobre o projeto

Este projeto foi desenvolvido com foco em simular um sistema embarcado simples, aplicando conceitos de:

- Arquitetura modular
- Separação de responsabilidades
- Simulação de sensores
- Processamento de dados em tempo real

---

## 🧩 Arquitetura

O sistema foi dividido em camadas:

- **Sensor** → Responsável pela leitura do peso
- **Classifier** → Aplica regras de negócio
- **Utils** → Formatação de saída
- **Main** → Orquestra o fluxo da aplicação

---

## ⚙️ Tecnologias

- Python 3
